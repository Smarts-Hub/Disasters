# Disasters - That trendy minigame!
Replica of Hypixel's new trendy mini-game.
 
## 📦 Installation

1. Download the plugin `.jar` file from [BuiltByBit](https://builtbybit.com/resources/disasters-that-trendy-mini-game.61539/).
2. Place the `.jar` file into the `plugins` folder of your Minecraft server.
3. Restart the server.
